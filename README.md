## 소개

목적지 근처의 공영 주차장 정보를 제공 뿐만 아니라 주차장에서 목적지까지 도보도 함께 책임지는 플랫폼입니다. 저희 서비스를 통해 단순한 주차장 위치 정보 뿐만 아니라 잔여 좌석, 예상 요금, 최종 목적지까지의 도보 경로 안내까지 경험해보세요!
